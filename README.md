# 📧 Email Reply AI Agent 🚀

An AI-powered email automation system built using n8n that processes incoming Gmail messages, generates human-like replies using LLM (Groq AI), and stores structured data in Airtable for tracking and workflow automation.

---

## ✨ Features
- 📩 Automatically reads incoming Gmail emails
- 🧠 Uses LLM (Groq AI) for smart reply generation
- ✍️ Generates professional human-like email responses
- 📊 Stores email data in Airtable for record keeping
- 🔄 Automates end-to-end email workflow

---

## 🧠 Tech Stack
- n8n (Workflow Automation)
- Gmail API
- Groq AI (LLM)
- Airtable
- LangChain AI Agent

---

## ⚙️ How It Works
1. Gmail receives an email
2. n8n fetches email data
3. AI Agent analyzes content using LLM
4. Generates professional reply
5. Stores data in Airtable
6. Marks email as processed
